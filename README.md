# AI 工具系统提示词展示

这是一个用于展示各种 AI 工具系统提示词、内部工具和模型配置的 Next.js 应用程序。

## 功能特点

- 按工具分类展示各种 AI 系统提示词
- 技术感与简约风格的 UI 设计
- 响应式布局，适配各种设备
- 直观的导航和内容展示

## 如何运行

确保已安装 Node.js (推荐 v16.x 或更高版本)

```bash
# 安装依赖
npm install

# 开发模式运行
npm run dev

# 构建生产版本
npm run build

# 运行生产版本
npm start
```

访问 http://localhost:3000 查看应用。

## 技术栈

- Next.js
- TypeScript
- Tailwind CSS
- React
